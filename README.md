Workout Tracker Bot
Телеграм-бот для записи ежедневных тренировок в локальную SQLite-базу.

Возможности:
Команда /start — создание базы и запуск бота.
Команда /workouts — просмотр всех тренировок с датами и днями недели.

Хранение данных в WorkoutsDataBase.sql.

Технологии
Python
Telebot (pyTelegramBotAPI)
SQLite
datetime
